## 2. Typography

- **Headings / numerals / roll numbers:** Fira Code (400–700) — tabular feel for data density
- **Body / UI:** Fira Sans (300–700)
- **Import:**
```css
@import url('https://fonts.googleapis.com/css2?family=Fira+Code:wght@400;500;600;700&family=Fira+Sans:wght@300;400;500;600;700&display=swap');
```
- Scale: page title 24/32 600; section title 18/28 600; body 14–16; table/dense data 13–14; caption 12.

## 3. Spacing

`--space-xs` 4px · `--space-sm` 8px · `--space-md` 16px · `--space-lg` 24px · `--space-xl` 32px · `--space-2xl` 48px · `--space-3xl` 64px.
Dashboards are data-dense: default card padding 16–24px, table cell padding 8–12px.

## 4. Radius & Shadows

- Radius: inputs/buttons `8px`, cards `12px`, modals `16px`, badges/pills `9999px`.
- Shadows: `--shadow-sm 0 1px 2px rgba(0,0,0,.05)` (subtle lift) · `--shadow-md 0 4px 6px rgba(0,0,0,.1)` (cards/buttons) · `--shadow-lg 0 10px 15px rgba(0,0,0,.1)` (modals/dropdowns) · `--shadow-xl 0 20px 25px rgba(0,0,0,.15)` (featured).

## 5. Component Specs

- **Primary button:** accent `#D97706` bg, white text, 12×24 padding, radius 8, weight 600, 200ms ease, hover opacity .9 + translateY(-1px). `cursor-pointer`.
- **Secondary button:** transparent bg, `#1E40AF` text, 2px `#1E40AF` border, same geometry.
- **Card:** white surface, radius 12, padding 24 (16 in dense grids), `--shadow-sm`→`--shadow-md` on hover (no layout-shifting scale).
- **Input/Select:** 12×16 padding, 1px `#E2E8F0` border, radius 8, 16px font; focus: border `#1E40AF` + `0 0 0 3px #1E40AF20`, no bare `outline:none` without replacement.
- **Modal:** white, radius 16, padding 32, `--shadow-xl`, overlay `rgba(0,0,0,.5)` + 4px blur, max-width 500–640px.
- **Tables:** row hover highlight `#F1F5F9`, sticky header on long rosters, 13–14px, filtering always available on data tables.
- **Badges:** status tokens above, pill radius, 12px 500.

## 6. Motion

Hover tooltips, row highlighting, smooth filter animations, loading spinners. Transitions 150–300ms.
**All animation must respect `prefers-reduced-motion: reduce`** (disable non-essential transforms).
Framer Motion (Aether UI pieces): fades/short slides only — no bounce, no parallax, nothing flashy.

## 7. Anti-Patterns (Do NOT use)

- ❌ Ornate/flashy/neon design — this is an official college system
- ❌ Data tables without filtering
- ❌ Emojis as icons — use SVG (Lucide) only
- ❌ Missing `cursor-pointer` on clickables
- ❌ Layout-shifting hover transforms
- ❌ Text contrast below 4.5:1
- ❌ Instant state changes (always 150–300ms transitions)
- ❌ Invisible focus states
- ❌ Dark-mode-only styling (light mode is default)
